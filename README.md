CodeAlpha Task 2 - Exploratory Data Analysis (EDA)

Project Title

Exploratory Data Analysis Using Python

Objective

The objective of this project is to perform Exploratory Data Analysis (EDA) on a sample sales dataset. The analysis helps understand the dataset structure, identify trends and patterns, detect data quality issues, and generate meaningful business insights.

Tools Used

- Python
- Pandas
- Online Python Compiler
- GitHub

How to Run

1. Open an Online Python Compiler.
2. Copy and paste the Python code into the editor.
3. Run the program.
4. The script will perform Exploratory Data Analysis on the dataset.
5. View the results displayed in the output console.

Output

- Displayed the first five rows of the dataset.
- Analyzed dataset shape and dimensions.
- Checked data types of all columns.
- Identified missing values and duplicate records.
- Generated descriptive statistics.
- Calculated total, average, maximum, and minimum sales.
- Extracted useful business insights from the dataset.

Key Results

- Total Sales: 190000
- Average Sales: 38000.0
- Maximum Sales: 55000
- Minimum Sales: 20000
- Missing Values: 0
- Duplicate Rows: 0
